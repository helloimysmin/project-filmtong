# :tv: 프로젝트 명 : FILMTONG 
- 필름으로 통하는 우리
- 영화 소개/추천/리뷰 사이트


## :clapper: 프로젝트 개요
- 영화는 흔히들 종합 예술이라고 합니다.

  영화를 보며 간접 체험을 통해 희노애락을 느끼기도 하고 인생의 교훈 또한 얻게됩니다.
  
  따라서 이런 영화를 추천해주고 사람들이 서로의 리뷰를 나누며 
  
  영화를 공유할 수 있는 사이트를 만들었습니다.
